Tags vistas na aula

Tags h1-h6
Os elementos HTML h1-h6 representam seis níveis de título de seção. h1 é o nível de seção mais alto e h6 é o mais baixo.
Tag p
O elemento HTML p representa um parágrafo.
Tag br
O elemento HTML quebra-de-linha br produz uma quebra de linha em um texto
Tag DOCTYPE
No HTML, o doctype é a introdução
Tag html
O elemento html (ou HTML root element) representa a raiz de um HTML
Tag body
O elemento body do HTML representa o conteúdo de um documento HTML
style
O elemento HTML style contém informações de estilo para um documento
Tag script
O elemento HTML script é usado para incluir ou referenciar um script executável
Tag img
O elemento img (or HTML Image Element) representa a inserção de imagem no documento
Tag ul
O elemento HTML ul (ou elemento HTML de Lista desordenada) representa uma lista de itens sem ordem rígida
Tag ol
O Elemento HTML ol (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados
Tag li
O elemento HTML li (ou a Lista dos Itens de um elemento HTML) é usado para representar um item que faz parte de uma lista
Tag dl
O elemento HTML Definition List dl engloba uma lista de pares de termos e descrições
Tag dt
O elemento HTML dt (ou Elemento HTML de Definição de Termo) identifica um termo na lista de definição
Tag dd
O elemento HTML dd fornece detalhes ou uma definição mais completa do termo precedente (definido por dt) numa lista de descrições (dl).
